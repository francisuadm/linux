# linux


#### To generate a report of your hardward use the command below:

````
sudo lshw -html > lshw.html
````


#### How to view linux CPU information

````
sudo lscpu
````


#### How to view USB controller in Linux

````
sudo lsusb
````


#### How to view PCI controller in Linux

````
sudo lspci
````
